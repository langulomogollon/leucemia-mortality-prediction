# 🔬 Leucemia Mortality Prediction using Machine Learning

## 📌 Overview

This project applies Machine Learning techniques to predict mortality risk in leukemia patients using clinical and demographic variables.

The objective is to identify patterns associated with patient outcomes and evaluate predictive models that can support healthcare decision-making.

---

## 🎯 Objectives

- Analyze leukemia patient data.
- Identify factors associated with mortality.
- Train and evaluate classification models.
- Compare model performance using standard metrics.

---

## 🛠️ Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Jupyter Notebook

---

## 📊 Methodology

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Training
5. Model Evaluation

---

## 📈 Results

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score
